LayoutGrid Component synced with Web Awesome 3.0.0

provides layout grid functionality for Angular applications.

Preparing for Web Awesome 1.0.0. This wrapper will not target a version above 1.0.0 until it is released.

Overview

Import

Import the layoutGrid directive from the angular-awesome package:

// Import the layoutGrid directive
import { WaLayoutGridDirective } from 'angular-awesome/layout-grid';

Examples

Grid Directive Examples

Use [waGrid] to apply the Web Awesome grid primitive with typed options for sizing. Combine with [waGap] for spacing. Use [waGridSpan] on a child to span all columns.
