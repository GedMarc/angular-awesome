LayoutFrame Component synced with Web Awesome 3.2.x

provides layout frame functionality for Angular applications.

Overview

Import

Import the layoutFrame directive from the angular-awesome package:

// Import the layoutFrame directive
import { WaLayoutFrameDirective } from 'angular-awesome/layout-frame';

Examples

Frame Directive Examples

Frames are well-suited for images and image placeholders. Use [waFrame] to get typed hints and convenient options.

Basic frame

<div waFrame style="max-inline-size: 20rem;">
  <div></div>
</div>
