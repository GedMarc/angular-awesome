Text Component synced with Web Awesome 3.0.0

provides text functionality for Angular applications.

Preparing for Web Awesome 1.0.0. This wrapper will not target a version above 1.0.0 until it is released.

Overview

Import

Import the text directive from the angular-awesome package:

// Import the text directive
import { WaTextDirective } from 'angular-awesome/text';
