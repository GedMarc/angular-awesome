LayoutFlank Component synced with Web Awesome 3.2.x

provides layout flank functionality for Angular applications.

Overview

Import

Import the layoutFlank directive from the angular-awesome package:

// Import the layoutFlank directive
import { WaLayoutFlankDirective } from 'angular-awesome/layout-flank';

Examples

Flank Directive Examples

Use the [waFlank] directive to turn any container into a Web Awesome flank. Combine it with [waGap] for spacing and [waAlignItems] for alignment.

Basic flank

<div waFlank>
  <div></div>
  <div></div>
</div>

Inputs with adjacent button (end flank)

<div waFlank [waFlankPosition]="'end'" [waGap]="'xs'">
  <wa-input>
    <wa-icon slot="start" name="magnifying-glass"></wa-icon>
  </wa-input>
  <wa-button>Search</wa-button>
</div>
